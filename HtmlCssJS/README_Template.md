## Project Overview

> This project is a front-end based Restaurant Management System developed using HTML, CSS, and JavaScript.  
It allows users to browse menu items, add items to a cart, and simulate order placement.  
The interface is designed to be user-friendly and responsive across different devices.  
This project focuses on implementing core frontend concepts without backend integration.  
The main goal is to understand DOM manipulation, event handling, and responsive design.
