# 1. Course Introduction

The course intends to enable the students to be acquainted with the basic concepts of programming methodology, ‘C’ Programming language.

---

# 2. Objectives

At the end of this course the students should be able:

- To develop a programming logic.
- To teach basic principles of programming.
- To develop skills for writing programs using ‘C’.

---

# 3. Specific Objectives and Contents

## Specific Objectives

- Define algorithm, use of algorithms
- Describe different notations of algorithms
- State standard notations and common functions
- Classify different Pseudo-code Conventions
- Develop fundamental algorithms
- Write different algorithms for different problems
- Differentiate different programming approaches and their benefits.
- Understand the basic structure of C program
- Understand different types of data types and qualifiers in terms of memory requirement and range.
- Write various programs using different data types, qualifiers.

# Unit I: Introduction To Algorithms and C (8 Hrs)

## Fundamentals of algorithms

Notion of an algorithm. Pseudo-code conventions like assignment statements and basic control structures.

### Algorithmic problems

Develop fundamental algorithms for:

1. Exchange the values of two variables with and without temporary variable
2. Counting positive numbers from a set of integers
3. Summation of set of numbers
4. Reversing the digits of an integer
5. Find smallest positive divisor of an integer other then 1
6. Find G.C.D. and L.C.M. of two as well as three positive integers
7. Generating prime numbers

## Different approaches in programming

- Procedural approach
- Object Oriented approach
- Event Driven approach

## Structure of C

Header and body, Use of comments, Compilation of program.

## Data Concepts

Variables, Constants, data types like:

- int
- float
- char
- double
- void

## Qualifiers

Short and ling size qualifiers, signed and unsigned qualifiers.

Declaring variables.

Scope of the variables according to block.

Hierarchy of data types.
