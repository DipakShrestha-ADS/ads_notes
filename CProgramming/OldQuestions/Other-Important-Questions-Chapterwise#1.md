# **Important Questions for Exams**  

### Unit I: Introduction To Algorithms and C (8 Hrs)
1. Define an algorithm. List and explain any five of its important characteristics. (Very Short – 2 marks)  
2. What is the difference between algorithm and flowchart? Draw a flowchart to find the G.C.D. of two positive integers. (Short – 6 marks)  
3. Develop an algorithm (using pseudo-code) to:  
   (a) Reverse the digits of an integer  
   (b) Generate all prime numbers up to N.  
4. Write a C program to exchange the values of two variables (i) with a temporary variable and (ii) without a temporary variable.  
5. Differentiate between Procedural, Object-Oriented and Event-Driven programming approaches with examples.

### Unit II: Basic of C (4 Hrs)
6. Explain the different data types available in C programming with their memory size and range. (Short – 6 marks)  
7. What is the difference between syntax errors and logical errors in C programming? Give one example of each. (Very Short – 2 marks)  
8. Describe the various storage classes in C (auto, extern, static, register) with suitable examples.  
9. Write a C program that inputs the radius of a circle and calculates and displays its area and circumference. (Repeated in 2081)  
10. Explain the use of break and continue statements in loops with examples.

### Unit III: Arrays, Strings and Sorting Techniques (8 Hrs)
11. What is an array? Differentiate between 1-D and 2-D arrays with declaration and initialization syntax.  
12. Write a C program to check whether an entered string is a palindrome or not. (Repeated in 2082)  
13. Write a menu-driven C program to perform the following operations on a 1-D array:  
    (i) Insert an element (ii) Delete an element (iii) Display the array.  
14. Explain Bubble Sort and Selection Sort algorithms. Write a C program to implement Bubble Sort on an array of 10 integers.  
15. Describe any three string manipulation functions (strlen, strcpy, strcat, strcmp) with examples.

### Unit IV: Functions, Storage Classes and Recursion (8 Hrs)
16. Discuss the concept of functions in C. Write a program demonstrating user-defined functions with return values. (Repeated in 2082)  
17. Differentiate between call by value and call by reference with suitable examples. (Short – 6 marks)  
18. What is recursion? Write a recursive C program to find the factorial of a given number. (Repeated pattern in 2081)  
19. Explain storage classes (Automatic, External, Static, Register) and their scope, visibility and lifetime.  
20. Write a C program to display the Fibonacci series up to n terms using recursion.

### Unit V: Structure and Union (4 Hrs)
21. What is a structure in C? How does it differ from an array? (Very Short – 2 marks)  
22. Write a C program to define a structure for a student (name, roll-number, marks in three subjects). Accept details for one student and display the total and average marks. (Repeated in 2082)  
23. Explain nested structure with an example.  
24. Define union. Write a program to demonstrate the difference between structure and union.

### Unit VI: Pointers and File Handling (6 Hrs)
25. What is a pointer? Explain its uses with suitable examples. (Repeated in 2081)  
26. Write a C program to find the entered number from a given list with its position using pointers. (Similar to 2082 Q11)  
27. Explain dynamic memory allocation functions: malloc(), calloc(), realloc() and free().  
28. What are file opening modes in C? Write a C program to create a new file, read name, address and marks of n students from the user and write them to the file. (Repeated pattern in 2081 Q18)

### Unit VII: Link Lists (4 Hrs)
29. Define a linear linked list. State its advantages and disadvantages over an array.  
30. Write algorithms (in pseudo-code) for:  
    (a) Insertion at the beginning of a linked list  
    (b) Deletion of a node from a linked list.  
31. Write a complete C program to implement a singly linked list with insert and delete operations.

### Unit VIII: Stacks (3 Hrs)
32. Define a stack. Write algorithms for push and pop operations using an array.  
33. Write a C program to implement a stack (using array) that supports push, pop and display operations.

### Unit IX: Queues (3 Hrs)
34. Define a queue. Differentiate between straight queue and circular queue.  
35. Write algorithms for insertion and deletion in a queue. Implement a queue using an array in C.

**Long Answer / Program-heavy Questions**  

36. Write a C program to implement a simple calculator using switch-case statement. Explain the logic used.
37. Write a C program to store n numbers in a list and search whether the entered number is found in the list or not. 
38. Write a recursive C program to find the factorial of an entered number and also explain the components of a function.
39. Create a structure named “book” with members name, author and publisher. Write a C program to read data of 50 books and display the names of books published by “XYZ” publisher.
