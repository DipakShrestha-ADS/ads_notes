# **Important Questions for Exams** 

**Unit I: Introduction To Algorithms and C**  

1. What is the notion of an algorithm? Explain its use in programming.  
2. Write pseudo-code for summation of a set of numbers.  
3. Develop an algorithm to find the smallest positive divisor of an integer other than 1.  
4. Explain different notations used for representing algorithms.  
5. Write an algorithm to reverse the digits of a given integer.  
6. Differentiate between event-driven and procedural programming approaches.

**Unit II: Basic of C**  

7. Explain the hierarchy of data types in C.  
8. Describe formatted input/output functions with examples (printf and scanf).  
9. What are the different types of operators in C? Explain arithmetic and relational operators.  
10. Explain type conversion (automatic and explicit) in C with examples.  
11. Write a C program to demonstrate the use of increment and decrement operators.  
12. Explain the scope of variables according to block in C.

**Unit III: Arrays, Strings and Sorting Techniques** 

13. How do you declare and initialize a one-dimensional array in C? Give example.  
14. Write a C program to find the largest and smallest element in an array.  
15. Explain insertion sort algorithm and implement it in C.  
16. Write a C program to sort an array using merge sort.  
17. Describe character and string handling functions with examples.  
18. Write a C program to demonstrate array of strings.

**Unit IV: Functions, Storage Classes and Recursion** 

19. Explain global and local variables in functions with examples.  
20. What is the difference between functions and macros in C?  
21. Write a C program to demonstrate passing arrays to functions.  
22. Explain the advantages of recursion with suitable example.  
23. Write a recursive C program for Tower of Hanoi problem.  
24. Explain the concept of scope, visibility and longevity of variables.

**Unit V: Structure and Union**  

25. Explain array of structures and arrays within structures with examples.  
26. Write a C program to demonstrate structures and functions.  
27. Explain how to read and assign values to structure variables.  
28. Write a program using union to store different types of data.

**Unit VI: Pointers and File Handling** 

29. Explain pointer arithmetic and chain of pointers with examples.  
30. Write a C program to demonstrate pointers and arrays.  
31. Explain pointers as function arguments and functions returning pointers.  
32. Write a C program using pointers to swap two numbers.  
33. Explain text and binary files in C with examples.  
34. Write a C program to read from a file and display its contents.

**Unit VII: Link Lists**  

35. Explain representation of linked list in memory.  
36. Write an algorithm for searching a particular node in a linked list.  
37. Write an algorithm for deletion of a node from a linked list.  
38. Differentiate between linked list and array with advantages and shortcomings.

**Unit VIII: Stacks**  

39. Explain array representation of stack.  
40. Write a C program to implement stack with push, pop and display operations.  
41. Differentiate between stack and array.

**Unit IX: Queues**  

42. State the applications that use queues.  
43. Differentiate between straight queue and circular queue.  
44. Implement queue using linked list in C.  
45. Write algorithms for basic operations of a queue.

**Long Answer Type Questions**  

46. Write a C program to enter numbers until user presses 0 and display sum and average of entered numbers.  
47. Write a C program to perform various arithmetic operations on two operands using switch case.  
48. Write a C program to store n numbers in a list and check whether the entered number is found in the list or not.  
49. Explain components of functions. Write a C program to display factorial using recursion.  
50. What are file opening modes? Write a program to open a new file and write student details into it.
