# **Important Questions for Exams**  

**Unit I: Introduction To Algorithms and C**  

1. Explain the notion of an algorithm and its uses in programming.  
2. Write pseudo-code conventions for basic control structures with examples.  
3. Develop an algorithm to find G.C.D. and L.C.M. of three positive integers.  
4. Explain different programming approaches and their benefits.  
5. Write an algorithm to generate prime numbers up to a given limit.  
6. Describe the structure of a C program including header and body.

**Unit II: Basic of C**  

7. Explain different types of operators in C with suitable examples.  
8. Describe formatted and character I/O functions in C with examples.  
9. What is type conversion? Explain automatic and explicit type conversion.  
10. Write a C program to demonstrate compound assignment and conditional operators.  
11. Explain the working of decision making statements: if, if-else and switch.  
12. Describe jump statements: break, continue and goto with examples.

**Unit III: Arrays, Strings and Sorting Techniques**  

13. Explain one-dimensional and two-dimensional arrays with declaration and initialization.  
14. Write a C program to add two matrices using 2-D arrays.  
15. Describe insertion sort and merge sort algorithms with their efficiency.  
16. Write a C program to count frequency of each character in a string.  
17. Explain string handling functions and write a program to concatenate two strings without using library function.  
18. Write a C program to sort an array in descending order using selection sort.

**Unit IV: Functions, Storage Classes and Recursion**  

19. Explain global and local variables along with function definition and return statement.  
20. Differentiate between functions and macros with examples.  
21. Write a C program to pass array as argument to a function.  
22. Explain recursion with advantages and write a program for Fibonacci sequence.  
23. Describe storage classes and their scope, visibility and lifetime.  
24. Write a C program using recursion for Tower of Hanoi.

**Unit V: Structure and Union**  

25. Explain array of structures and structures within structures with examples.  
26. Write a C program to demonstrate passing structure to a function.  
27. Explain how to assign and read values from structure variables.  
28. Write a program to show the use of union for storing different data types.

**Unit VI: Pointers and File Handling**  

29. Explain pointer arithmetic, chain of pointers and pointers to arrays.  
30. Write a C program to demonstrate pointers as function arguments.  
31. Explain dynamic memory allocation with malloc(), calloc(), realloc() and free().  
32. Write a C program using pointers to copy one string to another.  
33. Describe text and binary files and write a program to append data to a file.  
34. Explain fseek() and other file handling functions with examples.

**Unit VII: Link Lists** 

35. Explain representation of linear linked list in memory.  
36. Write an algorithm for traversing and searching a node in a linked list.  
37. Write a C program to insert a node after a given node in a singly linked list.  
38. Differentiate between linked list and array with advantages and shortcomings.

**Unit VIII: Stacks**  

39. Explain array representation of stack and its features.  
40. Write algorithms for basic operations (push and pop) on stack.  
41. Write a C program to implement stack for checking balanced parentheses.

**Unit IX: Queues**  

42. Explain applications of queues and basic operations.  
43. Differentiate between straight queue and circular queue.  
44. Write algorithms for insertion and deletion in a circular queue.  
45. Implement a queue using linked list in C.

**Long Answer Type Questions** 

46. Write a C program to enter numbers until user presses 0 and display sum and average.  
47. Write a menu-driven C program to perform arithmetic operations on two operands.  
48. Write a C program using structure to store and display book details.  
49. Explain components of functions and write a program to calculate factorial using recursion.  
50. What are different file opening modes? Write a program to read student data and write to a binary file.
